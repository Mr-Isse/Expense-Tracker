# Expense-Tracker
Expense Tracker waa Full Stack Application casri ah oo aan anigu sameeyay, looguna talagalay in lagu maareeyo kharashaadka (expenses) si fudud, degdeg ah, oo muuqaal qurux badan leh. App-kan wuxuu ka caawinayaa user-ka inuu la socdo dakhliga iyo kharashaadka maalinlaha ah, isla markaana uu arko warbixin cad (reports &amp; analytics).
